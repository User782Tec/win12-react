## 安装和使用

1. 确保计算机安装了Node.js(>=20.10.0)并能正常使用node、npx和npm命令 

2. 下载本项目代码（可以使用git clone或直接下载项目压缩包后解压）。

3. 进入项目根目录，运行：`npm install`安装项目依赖。

4. 运行`npm run start`后即可搭建本地服务器。打开浏览器并访问`localhost:3000`来体验或开发。
